# Devpost Submission Handoff

## Tagline

**Practice the judgment that comes after AI writes the code.**

## One-paragraph description

Pager is an execution-verified incident simulator for developers working beside AI. Learners enter a realistic production incident, inspect a real codebase and operational signals, judge plausible AI repair proposals, and prove their choice against a deterministic acceptance suite. The product intentionally treats AI guidance as a hypothesis—not an authority—so the credential is earned only through executable evidence.

## Technical highlights

- Next.js App Router, TypeScript, Monaco, WebContainers, and Pyodide
- Manifest-owned incident artifacts, telemetry, briefs, and stakeholder context
- Optional OpenAI Coach for scoped questions; authored repair options and deterministic execution remain the learning and grading authority
- Five execution-verified incidents across easy-to-advanced Python and TypeScript practice paths
- Isolated candidate proof scripts for both languages

## Honest limitations

- Java and C++ are intentionally not advertised until isolated compiler sandboxes are provisioned.
- The optional AI Pair is advisory and should not receive credentials or customer data.
- Browser smoke testing on the deployed URL is a final release step before submission.
