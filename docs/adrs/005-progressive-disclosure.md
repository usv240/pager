# ADR 005: One workspace, progressive disclosure

## Decision
Pager keeps code in the center, incident context on the left, and conversation plus repair review on the right. Panels are resizable and guided practice explains them in sequence.

## Why
This resembles how engineers investigate a real incident while keeping the core path understandable for learners.

## Tradeoff
The workspace has more information than a coding exercise. The guide, focus controls, and tabs reduce that cognitive load.
